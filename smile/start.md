I. Các câu lệnh trong git
1. git add .
2. git commit -m "fix bug"
3. git push origin main
4. git init:  là lệnh dùng để khởi tạo một kho lưu trữ (repository - repo) Git mới, cục bộ (local) trong thư mục dự án hiện tại
5. git clone: lấy từ trên mạng về
6. git pull: đồng bộ từ trên mạng về máy
7. git add: thêm file, folder
8. git commit: sửa
9. git push: đồng bộ từ máy lên repo mạng
10. git log hoặc git status: check trạng thái của file
11. git checkout: chuyển giữa các branch
12. git merge: gộp code từ các branch lại với nhau
13. git branch: tạo branch mới
14. git branch -d: xóa branch
15. git branch -m: đổi tên branch
16. git branch -a: liệt kê tất cả các branch
17. git branch -r: liệt kê các branch trên mạng
18. git branch -vv: liệt kê các branch trên mạng và cục bộ
19. git branch -u origin/main: đồng bộ branch trên mạng và cục bộ
20. uoe
